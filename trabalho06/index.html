Manual de Instalação

Siga os passos abaixo para configurar o ambiente e executar a sua Rede Social.

01. Preparação e Arquivos
  • Abra a pasta do DRIVE e atualize seus arquivos: Pasta do drive
  • Abra a pasta do seu projeto no VSCode[cite: 3]. Caminho sugerido: Documentos/seu_nome/sistema06/[cite: 4]. Pode usar o clique direito do mouse e selecionar "Abrir com Code"[cite: 5].
  • Acesse o sistema atualizado na pasta compartilhada do Drive[cite: 2, 6]. Atualize TODOS os arquivos copiando o conteúdo para o seu projeto local[cite: 6].
  • Crie os novos arquivos HTML estruturais dentro da pasta templates/[cite: 7]:
    • rede.html [cite: 8]
    • rede_login.html [cite: 9]
  • Dentro da pasta static/config/, crie o arquivo de estilização[cite: 10]:
    • rede.css [cite: 11]
  • Confirme se o arquivo principal sistema.py está devidamente atualizado[cite: 12].
02. Configuração do Banco de Dados
  • Abra o pgAdmin [cite: 13] e crie um novo banco de dados chamado redeX[cite: 14].
  • Abra a ferramenta de Query e execute os comandos SQL abaixo, bloco por bloco[cite: 15]:
-- 1. Tabela de Usuários
create table usuarios(
    id serial primary key,
    username text,
    senha text
);

-- 2. Tabela de Postagens
create table posts(
    id serial primary key,
    conteudo text,
    id_autor int,
    foreign key (id_autor) references usuarios(id)
);

-- 3. Tabela de Curtidas
create table curtida(
    id serial primary key,
    id_curtiu int,
    id_post int,
    foreign key (id_curtiu) references usuarios(id),
    foreign key (id_post) references posts(id)
);

-- 4. Inserindo Usuário Administrador (Senha com Hash criptografado)
insert into usuarios (username, senha) values ('admin', 'sutvegfhacnp132435');

-- 5. Inserindo Postagem de Teste
insert into posts (conteudo, id_autor) values ('Testandooooo0', 1);
💡 Nota sobre Segurança: Repare que no comando INSERT do administrador, a senha original stefan123 foi inserida criptografada usando um código de caracteres aleatórios (Hash)[cite: 34, 38, 42].
03. Execução do Sistema & Teste de Login
  • No VSCode, abra o terminal integrado pressionando as teclas Ctrl + J[cite: 39].
  • Inicie o servidor digitando o comando: python sistema.py e aperte Enter[cite: 39].
  • Abra o Google Chrome e digite diretamente na barra de endereços o local: localhost:5000 (não use a barra de pesquisas)[cite: 40].
  • Clique no botão para acessar a rede social [cite: 41] e realize o login com as credenciais padrão[cite: 42]:
    • User: admin [cite: 42]
    • Senha: stefan123 [cite: 42]
  • O sistema deverá carregar a interface da rede com a logo REDE X05[cite: 43, 48].
🎯 Desafio Prático: Use comandos SQL no pgAdmin para cadastrar o seu perfil e o de mais 2 amigos da turma[cite: 50, 51]. Utilize o programa gerador de Hash para embutir as senhas da forma correta[cite: 52].
04. Importação de Dados em Massa
  • Baixe o arquivo de dados chamado aluninhos_pirigosos.csv[cite: 55].
  • No pgAdmin, clique com o botão direito sobre a tabela posts e selecione a opção Import/Export data[cite: 56].
  • Deixe marcado o botão para Import [cite: 63], selecione o caminho do arquivo CSV baixado no campo Filename [cite: 57] e mude o Encoding para UTF8[cite: 70]. Clique em OK[cite: 57, 83].
  • Atualize a página do seu navegador para visualizar todas as novas publicações vinculadas aos usuários[cite: 84]. Tudo funcionando? Apresente o resultado para o professor[cite: 85]!
⚠️ Tratamento de Erros (Process Failed):
Se a importação falhar, clique no botão "View Processes" no balão de erro. Em seguida, clique no ícone de documento (ao lado do número do PID) para visualizar o relatório detalhado do erro apresentado pelo PostgreSQL[cite: 89, 91].
